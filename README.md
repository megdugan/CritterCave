## Contents

+ ```templates/``` contains html files for all webpages.
+ ```app.py``` is the main app, handling routing with Flask.
+ ```static/``` contains the styling template and website favicon.
+ ```crittercave.sql``` sql file for initializing database.
+ ```other .py files``` contain various database functions.

## Technology Stack
+ **Frontend:** Flask, HTMl, CSS, JavaScript
+ **Backend:** Python and MySQL

## Our Design Process-- Milestones and Next Steps

**Draft**
+ Build basic functionality while focusing on the individual profile: sign-up and sign-in, uploading critters and stories.

**Alpha**
+ Build out communal functionality: creating stories for other critters, searching for users and critters.

**Beta**
+ Finalize functionality and reach goals: liking, deleting and editing critters and stories.
